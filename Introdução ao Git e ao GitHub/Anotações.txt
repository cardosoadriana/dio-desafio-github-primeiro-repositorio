Comandos para o Git e GitHub:

cd - navegar entre as pastas
cd .. - volta para navegação da pasta anterior
dir - lista de diretórios dentro da pasta atual;
cls - limpar a tela
mkdir - criar uma nova pasta
del - remove apenas arquivos
rmdir - remove todo o diretório (pastas e arquivos)
openssl sha1: salvar alterações dos arquivos ( para uma nova hash)
